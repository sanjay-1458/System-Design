
# System Design Repository

A curated repository focused on learning and implementing real-world system design concepts using practical examples, architecture discussions, UML modeling, API design, and object-oriented design principles.  

This repository is divided into two major sections:

- **Low Level Design (LLD)**  
  Covers object-oriented design problems, UML diagrams, API modeling, lifecycle flows, and commonly used software design patterns implemented in C++.

- **High Level Design (HLD)**  
  Covers scalable distributed systems, architecture patterns, infrastructure-level decisions, scalability tradeoffs, reliability, and system architecture concepts.
